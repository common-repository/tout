Enables Tout's Javascript SDK on all pages.

== Description ==

This plugin adds the required Javascript for Tout's Javascript SDK.

For more information on Tout visit:

[Tout](http://www.tout.com)

For more information on the Tout SDK, please contact your Tout account representative.

== Installation ==

1. Upload `tout` directory to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Set the Tout Property ID that Tout provided to you under the Tout settings (Admin > Settings > Tout)

4. If you are using the Tout Top Article auto-matching technology and you need to run some custom
   Javascript when a top-article match exists, provide that Javascript code in the settings as well.

4. If you are using the Tout Mid Article auto-matching technology and you need to run some custom
   Javascript when a mid-article match exists, provide that Javascript code in the settings as well.
